# 安装时出现

The 'PlatformIO IDE' extension is not available in Visual Studio Code for Mac Silic。

在MAC系统，可进入platform所在目录对应的Python虚拟环境：
source ~/.platformio/penv/bin/activate
执行如下安装命令
pip3 install cryptography
成功后即可再次进入VSCode的扩展插件安装。
